# 📖 About

This project is a full-stack blog application developed using Django. It allows users to browse blog posts through a clean and responsive interface while demonstrating Django's MVC architecture, template rendering, URL routing, and database integration.

The application was built to strengthen backend development skills and showcase Django best practices.

---

# ✨ Features

- 📝 Blog Management
- 📚 View Blog Posts
- 🎨 Responsive UI
- ⚡ Fast Django Backend
- 🗄️ SQLite Database
- 📱 Mobile Friendly
- 🚀 Deployed on Render

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/kulkarnicodes-dev/blogsite.git
```

## Navigate

```bash
cd blogsite
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Migrations

```bash
python manage.py migrate
```

## Start Server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000/
```
